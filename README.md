# Encrypt It

A simple command line program that will create a password for you, and then give you the option to encrypt it using various methods.

## Installation
```python
pip install -r requirements.txt
```

## Example Syntax

- [x] Base64
- [x] Base32
- [x] Reverse Cipher
- [x] Fernet Encryption
