# encrypt-it

A simple command line program that will create a password for you, and then give you the option to encrypt it with a specific method.

Current supported encryptions:

- Base64
  - [x] Encoding
- Base32
  - [x] Encoding
- Caesar Cipher
  - [ ] Encoding
- ROT13 Algorithm
  - [ ] Encoding
- RSA Cipher
  - [ ] Encoding
- Vignere Cipher
  - [ ] Encoding
- Reverse Cipher
  - [x] Encoding

Pull requests are welcome! :)
